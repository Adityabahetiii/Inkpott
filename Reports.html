List

API endpoint:

POST https://phprank.lunatio.com/api/v1/reports

Request Example:
curl --location --request POST 'https://phprank.lunatio.com/api/v1/reports' \
--header 'Content-Type: application/x-www-form-urlencoded' \
--header 'Authorization: Bearer {api_key} \
--data-urlencode 'url={url}'
Parameter Type Description
url required string The webpage's URL.
privacy optional integer Report page privacy. Possible values are: 0 for Public, 1 for Private, 2 for Password. Defaults to: 0.
password optional string The password for the report page. Only works with privacy set to 2.
